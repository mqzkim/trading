### Broken / Missing (4 gaps)

1. **`proposal_gen_handler` — no trigger** (affects SELF-02/03/04)
   - `ctx["proposal_gen_handler"]` stored in bootstrap but never called

2. **`signal_direction` not populated** (affects PERF-03/04)
   - `Position.close()` constructs `PositionClosedEvent` with `signal_direction=""` always
   - Fix: add `signal_direction` field to `Position` entity, set from trade plan at open, pass through `PositionClosedEvent`

3. **`sentiment_confidence` not persisted** (affects DASH-01)
   - `scored_symbols` table missing column; dashboard always returns `"NONE"`
   - Fix: add SQLite migration, update `save()` and `find_all_latest_with_details()` in sqlite_repo.py

4. **Dashboard env var documentation gap** (affects DASH-02 operationally)
   - `NEXT_PUBLIC_API_URL` and `API_JWT_TOKEN` required but undocumented
   - Fix: add `dashboard/.env.example`
