    1. Add `WatchlistEntry` VO to `src/portfolio/domain/value_objects.py`:
       ```python
       @dataclass(frozen=True)
       class WatchlistEntry(ValueObject):
           symbol: str
           added_date: date = field(default_factory=date.today)
           notes: Optional[str] = None
           alert_above: Optional[float] = None
           alert_below: Optional[float] = None
           def _validate(self) -> None:
               if not self.symbol or not self.symbol.strip():
                   raise ValueError("Symbol must not be empty")
       ```

    2. Add `IWatchlistRepository` ABC to `src/portfolio/domain/repositories.py`:
       ```python
       class IWatchlistRepository(ABC):
           @abstractmethod
           def add(self, entry: WatchlistEntry) -> None: ...
           @abstractmethod
           def remove(self, symbol: str) -> None: ...
           @abstractmethod
           def find_all(self) -> List[WatchlistEntry]: ...
           @abstractmethod
           def find_by_symbol(self, symbol: str) -> Optional[WatchlistEntry]: ...
       ```
       Also update the `from .value_objects import ...` line to include WatchlistEntry. Update `src/portfolio/domain/__init__.py` to export IWatchlistRepository and WatchlistEntry.

    3. Create `src/portfolio/infrastructure/sqlite_watchlist_repo.py`:
       - `SqliteWatchlistRepository(IWatchlistRepository)`
       - Constructor takes `db_path: str = "data/portfolio.db"` (same DB as positions).
       - Creates `watchlists` table: symbol TEXT PRIMARY KEY, added_date TEXT, notes TEXT, alert_above REAL, alert_below REAL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP.
       - `add(entry)`: INSERT OR REPLACE.
       - `remove(symbol)`: DELETE WHERE symbol=?.
       - `find_all()`: SELECT * ORDER BY added_date DESC, convert rows to WatchlistEntry VOs.
       - `find_by_symbol(symbol)`: SELECT WHERE symbol=?.
       Follow exact pattern from SqlitePositionRepository.

    4. Add CLI commands to `cli/main.py`:

       **dashboard** command (no arguments, option --portfolio-id defaults to "default"):
       - Instantiate `SqlitePositionRepository()` and `SqlitePortfolioRepository()`.
       - Load portfolio by id. Load all open positions via `find_all_open()`.
       - Render a `Panel` header showing: portfolio value (or initial_value if no positions), drawdown percentage, drawdown level (color-coded: normal=green, caution=yellow, warning=red, critical=bold red).
       - Render a `Table` with columns: Symbol, Qty, Entry Price, Market Value, Stop Price, Sector, Strategy.
       - If no positions, print "[dim]No open positions.[/dim]".
       - Use existing Rich patterns from the `analyze` command.

       **screener** command (options: --top-n default 20, --min-score default 60, --signal default "BUY", --output default "table"):
       - Instantiate `DuckDBSignalStore(duckdb.connect("data/analytics.duckdb"))`.
       - Call `query_top_n(top_n=top_n, min_composite=min_score, signal_filter=signal)`.
       - Render a `Table` with columns: Rank, Symbol, Composite, Risk-Adj, Intrinsic Val, MoS%, Signal, Strength.
       - If --output=json, print JSON.
       - If no results, print "[dim]No stocks match criteria.[/dim]".

       **watchlist_add** command (argument: symbol, options: --notes, --alert-above, --alert-below):
       - Instantiate `SqliteWatchlistRepository()`.
       - Create `WatchlistEntry` VO and call `repo.add(entry)`.
       - Print confirmation.

       **watchlist_remove** command (argument: symbol):
       - Instantiate `SqliteWatchlistRepository()`.
       - Call `repo.remove(symbol.upper())`.
       - Print confirmation.

       **watchlist_list** command (no arguments):
       - Instantiate `SqliteWatchlistRepository()`.
       - Call `repo.find_all()`.
       - Render a `Table` with columns: Symbol, Added, Notes, Alert Above, Alert Below.
       - If empty, print "[dim]Watchlist is empty.[/dim]".

       CLI naming: use underscores in function names (typer converts to hyphens automatically). So `def watchlist_add(...)` becomes `trading watchlist-add`.
