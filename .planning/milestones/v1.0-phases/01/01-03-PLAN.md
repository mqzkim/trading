    1. Create src/data_ingest/infrastructure/pipeline.py:
       - class DataPipeline:
         - __init__(self, max_concurrent: int = 5, db_path: str = "data/analytics.duckdb"):
           - Creates asyncio.Semaphore(max_concurrent)
           - Instantiates: YFinanceClient(semaphore), EdgartoolsClient(semaphore), DuckDBStore(), QualityChecker(), AsyncEventBus()
           - Connects DuckDBStore to db_path
         - async ingest_ticker(self, ticker: str) -> DataQualityReport | None:
           - Fetch OHLCV via yfinance_client.fetch_ohlcv(ticker)
           - Validate OHLCV via quality_checker.validate_ohlcv(ticker, df)
           - If OHLCV quality fails: publish QualityCheckFailedEvent, log warning, return report
           - Fetch financials via edgartools_client.fetch_financials(ticker)
           - Validate financials via quality_checker.validate_financials(ticker, records)
           - If financials quality fails: still store OHLCV (partial data is better than none), log warning
           - Store OHLCV in DuckDB via duckdb_store.store_ohlcv()
           - Store financials in DuckDB via duckdb_store.store_financials()
           - Publish DataIngestedEvent(ticker, ohlcv_rows, financial_quarters)
           - Return quality report
         - async ingest_universe(self, tickers: list[str] | None = None) -> dict:
           - If tickers is None: get from UniverseProvider.get_universe()
           - Run asyncio.gather(*[self.ingest_ticker(t) for t in tickers], return_exceptions=True)
           - Collect results: succeeded (list), failed (list), errors (list)
           - Return summary dict with counts and details
           - DuckDB writes are batched: collect all data first, write in single transaction
         - async close(self):
           - Close DuckDB connection
