### Note on Data Architecture
Scoring and regime detection intentionally fetch live data rather than reading from DuckDB. DuckDB serves as analytics/backtesting storage. This is a design decision, not a bug. The integration "breaks" in flows 3-5 reflect this architecture -- each step works correctly in isolation but the pipeline is not a linear data-pass chain.
