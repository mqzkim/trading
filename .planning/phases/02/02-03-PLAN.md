    3. compute_ensemble(dcf_value, dcf_confidence, epv_value, epv_confidence, relative_value, relative_confidence) -> dict:
       - Compute raw weights: base_weight * confidence for each model
       - Normalize to sum to 1.0 (effective_weights)
       - If total raw weight <= 0: return intrinsic_value=0, confidence=0
       - Weighted intrinsic = sum(effective_weight * value)
       - Confidence = 0.6 * model_agreement + 0.4 * data_completeness
       - Model agreement: 1 - CV of positive model values (higher agreement = lower CV)
       - Data completeness: total_raw_weight / sum(base_weights)
       - Clamp confidence to [0, 1]
       - Return: intrinsic_value, confidence, effective_weights, model_agreement, data_completeness

    4. compute_margin_of_safety(intrinsic_mid, market_price, sector) -> dict:
       - MoS = (intrinsic_mid - market_price) / intrinsic_mid
       - Look up sector threshold from SECTOR_MOS_THRESHOLDS (default 0.20)
       - has_margin = (MoS >= sector_threshold)
       - Return: margin_of_safety, sector_threshold, has_margin
