   - `__init__(self, ctx: dict)` -- receives the full bootstrap context dict

     - `drawdown_pct`: portfolio drawdown as percentage

     - `equity_curve`: for v1, generate a simple list from trade history P&L accumulation (if no daily snapshot table exists, derive from trade_plan execution dates + P&L). If no data, return empty lists for dates/values.
     - `regime_periods`: from regime_repo, list of {start, end, regime} dicts for equity curve overlay
   - Handle gracefully when repos return empty data (new system with no trades yet)
