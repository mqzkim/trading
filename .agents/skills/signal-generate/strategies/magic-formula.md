# Magic Formula 방법론 인코딩 (Greenblatt)

## 핵심 원리

저평가된 우량기업을 체계적으로 발굴.
두 가지 지표만으로 순위 매김.

## 두 가지 지표

### Earnings Yield (수익 수익률)
```
Earnings Yield = EBIT / Enterprise Value
EV = 시가총액 + 순부채
```
높을수록 좋음 (저평가 = 높은 수익 수익률)

### Return on Capital (자본수익률)
```
ROC = EBIT / (Net Working Capital + Net Fixed Assets)
NWC = 유동자산 - 유동부채
NFA = 총자산 - 유동자산 - 무형자산
```
높을수록 좋음 (우량 = 높은 자본수익률)

## 순위 매김

1. 전체 유니버스를 Earnings Yield로 순위 (1=최고)
2. 전체 유니버스를 ROC로 순위 (1=최고)
3. 두 순위 합산
4. 합산 순위 상위 20-30개 선택

## 시그널 변환

```
합산 순위 상위 10%: BUY (score: 90-100)
합산 순위 상위 25%: BUY (score: 70-89)
합산 순위 중간 50%: HOLD (score: 30-69)
합산 순위 하위 25%: SELL (score: 0-29)
```
