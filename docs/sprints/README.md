# Sprint Gate Artifacts

이 디렉토리는 스프린트별 Gate 산출물을 저장한다.

## 파일 규칙

스프린트 `Sx`에 대해 아래 파일을 생성한다.

- `Sx-context.md`
- `Sx-skill-check.md`
- `Sx-agent-plan.md`
- `Sx-verify.md`
- `Sx-release-review.md`
- `Sx-retrospective.md`

필요 시 Gate별 상세 로그를 추가한다.

## 시작 방법

1. `TEMPLATE-gate.md`를 복사해 `Sx-context.md`부터 생성
2. Gate 진행 중 증거(명령/결과/결정)를 누락 없이 기록
3. `TEMPLATE-retrospective.md`로 회고 작성
4. `TEMPLATE-closeout.md`로 Full Close-Out 수행
