# SEO Optimizer Agent (3-2)

> Layer 3 — 콘텐츠 & 마케팅 에이전트군 | Tier: Balanced
> 위험 등급: Low | Phase 2 구현 예정

## 핵심 기능

- 타겟 키워드 분석 ("best AI tool for X")
- 메타 타이틀/디스크립션 자동 생성 및 최적화
- Schema.org 구조화 데이터 자동 삽입
- 내부 링크 전략 제안 및 적용
- 검색 의도 분석 (informational/transactional/navigational)
- 경쟁 페이지 분석

## SEO 점수 산출

| 항목 | 가중치 | 체크 내용 |
|------|--------|----------|
| 제목 태그 | 20% | 키워드 포함, 60자 이내, 고유성 |
| 메타 디스크립션 | 15% | 155자 이내, CTA 포함, 키워드 |
| 헤딩 구조 | 15% | H1 유일성, H2/H3 계층, 키워드 |
| 콘텐츠 품질 | 20% | 단어 수, 키워드 밀도, 가독성 |
| 구조화 데이터 | 15% | Schema.org 적용, 유효성 |
| 내부 링크 | 15% | 관련 페이지 연결, 앵커 텍스트 |

## IDEA 3 시너지

디렉토리 사이트의 수익은 SEO 트래픽에 100% 의존:
- 신규 리스팅 → 자동 메타 태그 + 구조화 데이터
- 월간 랭킹 모니터링 + 콘텐츠 업데이트 식별
