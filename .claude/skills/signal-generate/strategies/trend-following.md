# Trend Following 방법론 인코딩

## 핵심 원리

추세의 존재를 확인하고, 추세 방향으로 진입.
3가지 확인 신호 중 2개 이상 충족 시 시그널 발생.

## 3가지 확인 신호

### 1. MA Cross (이동평균 교차)
```
50-day MA > 200-day MA → 골든크로스 (상승추세)
50-day MA < 200-day MA → 데드크로스 (하락추세)
Price > 200-day MA → 장기 상승추세
```

### 2. ADX (추세 강도)
```
ADX > 25 → 추세 존재
ADX < 20 → 추세 없음 (횡보)
+DI > -DI → 상승 추세
-DI > +DI → 하락 추세
```

### 3. Breakout (돌파)
```
Price > 20일 최고가 → 상방 돌파
Price < 20일 최저가 → 하방 돌파
Volume > 50일 평균 * 1.5 → 돌파 확인
```

## 시그널 판정

| 충족 수 | 시그널 | 스코어 |
|---------|--------|--------|
| 3/3 | Strong BUY | 3/3 |
| 2/3 | BUY | 2/3 |
| 1/3 | HOLD | 1/3 |
| 0/3 | SELL/AVOID | 0/3 |

## 스탑로스

- 초기 스탑: ATR(21) x 3.0
- 트레일링: 최고가 - ATR(21) x 2.5
- 추세 전환 (데드크로스): 전량 청산

## 보유 기간

- 추세 지속 시: 무제한 (트레일링 스탑까지)
- 추세 종료 신호 시: 청산
- 최소: 2주 (스윙)
