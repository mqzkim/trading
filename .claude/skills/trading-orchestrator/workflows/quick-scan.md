# Quick Scan 워크플로우

## 호출
`/trading-orchestrator quick "AAPL MSFT GOOGL"`

## 실행 순서

```
[병렬] 각 종목별 data-ingest
         ↓
[병렬] 각 종목별 scoring-engine (3 agents)
         ↓
[순차] 비교 테이블 생성
```

## 출력 예시

```
╔═══════╦═════════╦═══════╦═══════╦═══════╦═════════╗
║ 종목  ║ 복합점수 ║ 기본적 ║ 기술적 ║ 센티  ║ 안전성  ║
╠═══════╬═════════╬═══════╬═══════╬═══════╬═════════╣
║ AAPL  ║  78.5   ║  82   ║  74   ║  71   ║ PASSED  ║
║ MSFT  ║  82.1   ║  85   ║  79   ║  78   ║ PASSED  ║
║ GOOGL ║  71.3   ║  76   ║  68   ║  65   ║ PASSED  ║
╚═══════╩═════════╩═══════╩═══════╩═══════╩═════════╝
```
