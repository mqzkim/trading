# Product Launch Agent (3-4)

> Layer 3 — 콘텐츠 & 마케팅 에이전트군 | Tier: Balanced
> 위험 등급: Medium (외부 노출) | Phase 2 구현 예정

## 핵심 기능

- 플랫폼별 런칭 카피 자동 생성 (제목, 태그라인, 설명)
- D-Day 체크리스트 관리
- 업보트/인게이지먼트 전략 제안
- 런칭 후 트래픽/전환 분석 리포트

## 플랫폼별 가이드

- **ProductHunt:** 태그라인 60자, 설명 260자, 이미지 필수
- **Hacker News:** Show HN 제목만, 기술적 가치 명확, 과장 금지
- **GeekNews:** 한국어, 기술 용어 영어 유지, 간결 스타일
