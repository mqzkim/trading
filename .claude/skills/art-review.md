## 수행 가능 작업

### 1. UI 레이아웃 리뷰
Unity UI 코드/프리팹을 분석하여 레이아웃 개선안을 제시합니다.
- Canvas/RectTransform 설정 검토
- Safe Area 대응 확인
- 터치 타겟 크기 검증 (최소 44x44dp)
- 해상도별 호환성 확인

### 2. 에셋 목록 관리
필요한 에셋을 추적하고 누락된 에셋을 식별합니다.
- 스프라이트, 파티클, 사운드 에셋 목록 관리
- 에셋 명명 규칙 검증
- 아틀라스 구성 최적화

### 3. 컬러 팔레트 검증
게임 내 색상의 접근성과 가독성을 검증합니다.
- WCAG 대비 비율 확인
- 색맹 대응 검토
- 테마별 색상 조합 검증

### 4. 반응형 레이아웃 검토
다양한 해상도와 노치/다이나믹 아일랜드 대응을 확인합니다.
- 16:9, 18:9, 19.5:9, iPad 4:3 비율 대응
- 노치/펀치홀/다이나믹 아일랜드 영역 회피
- 배너 광고 영역 확보

### 5. 애니메이션 설계
DOTween 기반 애니메이션 코드 작성을 지원합니다.
- 볼 이동 애니메이션 (이징, 속도, 경로)
- UI 전환 애니메이션
- 클리어 이펙트 연출
- 파티클 시스템 설정
