# 백테스트 검증 체크리스트 상세

## 1. 데이터 무결성 (4항목)

### 1.1 Point-in-Time 펀더멘탈
- 재수정(restated) 데이터가 아닌 최초 발표 데이터 사용
- 결산일이 아닌 발표일 기준으로 데이터 참조
- **위반 시**: Look-ahead bias → 백테스트 무효

### 1.2 생존자 편향 없는 유니버스
- 상장폐지 종목 포함
- 인수합병 종목 포함
- 과거 시점의 구성종목 사용 (현재 구성종목 아님)
- **위반 시**: 수익률 과대 추정 → 결과 신뢰 불가

### 1.3 기업행위 조정
- 주식 분할 반영
- 배당 재투자 가정 명시
- 합병/분사 처리
- **위반 시**: 수익률 계산 오류

### 1.4 충분한 히스토리
- 최소 10년 (2개 이상 시장 사이클)
- 금융위기(2008), 코로나(2020) 등 극단 시나리오 포함
- **위반 시**: 특정 환경에만 최적화된 전략 가능성

## 2. 방법론 (4항목)

### 2.1 Walk-Forward 또는 CPCV
- 단순 IS/OOS 분할 불충분
- Walk-Forward: 롤링 윈도우 학습 → 테스트
- CPCV: Combinatorial Purged Cross-Validation
- **위반 시**: 오버피팅 탐지 불가

### 2.2 현실적 리밸런싱 주기
- 일간 리밸런싱 = 비현실적 (거래비용 무시)
- 스윙: 월간, 포지션: 분기 리밸런싱
- **위반 시**: 거래비용 과소 반영

### 2.3 거래비용 포함
- 수수료: 종목당 $0.01/주 이상
- 스프레드: 소형주 0.1-0.5%
- 시장 충격: 대규모 주문 시 추가 비용
- **위반 시**: 실제 성과 대폭 하락 가능

### 2.4 용량 제약 고려
- 일간 거래량의 1-5% 이내 주문
- 소형주 전략: 용량 한계 명시
- **위반 시**: 실행 불가능한 전략

## 3. 통계 검증 (4항목)

### 3.1 PBO < 10%
- Probability of Backtest Overfitting
- CPCV 기반 계산
- 10% 초과 시: 오버피팅 가능성 높음

### 3.2 WFE > 50%
- Walk-Forward Efficiency = OOS Sharpe / IS Sharpe
- 50% 미만 시: IS 성과 재현 불가

### 3.3 파라미터 민감도
- 핵심 파라미터 ±10% 변경
- 성과 변동 < 20% 유지
- 급격한 변동: 특정 파라미터에 과의존

### 3.4 샤프비율 t-stat > 2.0
- t = Sharpe * sqrt(N_years)
- 2.0 미만: 통계적으로 유의하지 않음
- 우연히 달성 가능한 수준
